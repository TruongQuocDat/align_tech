# README

Please run step by step:

Install Gemfile
`bundle install`

Install Yarn
`yarn install`

Create DB
`rake db:create`

Migrate table
`rake db:migrate`

Create data seed
`rake db:seed`

Please open 2 terminal
 - 1 tab: `bin/webpack-dev-server`
 - 1 tab start server (for local development): `rails s`
